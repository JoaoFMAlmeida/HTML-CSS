ESTRUTURA HTML


!DOCTYPE html
Define o tipo de documento como HTML5
html
Representa a raiz de um documento HTML ou XHTML. Todos os outros elementos devem ser descendentes desse elemento.
head
Container que fica invisível na página e contem metadado, título da página, links de integração com CSS e/ou fontes do Google Fonts.

Tags que geralmente vão dentro dela:"link, meta, title",
body
Container onde fica todo o conteúdo que deve ser visível na página.
